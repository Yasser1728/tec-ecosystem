# Domains Identity Statement / بيان هوية الـ Domains

## English
According to the latest engineering policies, each Domain within this repository is strictly considered as an entry point and identity marker only. No business logic or data should reside within any Domain. Domains serve exclusively as identity gateways by design.

## العربية
وفقًا للسياسات الهندسية الجديدة، يُعد كل Domain في هذا المستودع بمثابة بوابة/هوية فقط، ولا يجوز أن يحتوي على أي منطق برمجي أو بيانات. تُخصص الـ Domains كمعرّفات وبوابات تعريف حصرًا بحسب التصميم.
