# Contributing — TEC Knowledge Pack (EN / AR)

Thank you for contributing! / شكراً لمساهمتك!

How to contribute / كيفية المساهمة

1. Fork the repository and create a feature branch:
   - git checkout -b docs/update-tec-<topic>
2. Keep content bilingual (English + Arabic).
3. Use the canonical URL map (docs/url_map.md) for any new route additions.
4. Create a clear PR with a short changelog entry.

Reviewers / المراجعون

- Check for accuracy of domain mappings, translations, and legal claims.
- Ensure no public exposure of curated/private catalogs; link to brokered flows.

Style / الأسلوب

- English: Clear, professional.
- Arabic: Modern fusha with concise UX-oriented phrasing.
